# Meal Lab Project 🍲
Ολοκληρωμένο σύστημα διαχείρισης συνταγών σε Java.

## Χαρακτηριστικά
**Μέρος Α (API Library):** Σύνδεση με το TheMealDB API, αναζήτηση συνταγών και λήψη τυχαίας συνταγής.
**Μέρος Β (GUI App):** Περιβάλλον Swing για αναζήτηση και διαχείριση λιστών Favorites/Cooked.
**Persistence:** Αυτόματη αποθήκευση δεδομένων σε τοπικό αρχείο JSON (meals_data.json).
**Testing:** Πλήρης έλεγχος λειτουργικότητας με JUnit 5.

## Τεχνολογίες
Java 17+, Maven, Gson, JUnit 5.
